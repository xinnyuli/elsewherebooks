# 书店记账本 - 使用指南


## 🚀 第一步：打开软件

1. **双击桌面上的图标**（或者找到 `book_price_gui.py` 文件双击）
2. 等几秒钟，程序会自动打开

### ⚠️ 重要提醒：网络连接

**软件需要联网才能正常工作！**

- 软件会自动从网上获取**汇率数据**
- 如果没有网络：
  - ✅ 可以录入**二手书**和**书阁书**（用澳元定价）
  - ❌ **不能**录入外币新书
  - 会显示 **"❌ 网络错误"** 或 **"⚠️ 无汇率数据"**

**看到网络错误怎么办？**
1. 检查电脑是否连上WiFi
2. 连上网后，点击右上角的 **↻** 按钮更新汇率
3. 看到 **"汇率已更新"** 就可以正常使用了

---

## 👤 第二步：登录你的名字

软件打开后，会看到一个登录窗口：

```
📚
书店记账本
请选择当值店长
```

**怎么做：**
- 如果你的名字在列表里 → 点击选择你的名字
- 如果你是新员工 → 直接输入你的名字
- 点击 **"开始值班"** 按钮

✅ **完成！** 现在你进入了主界面

---

## 📖 第三步：录入卖出的书

### 情况1️⃣：卖 **新书**

1. 点击左下角的 **📕 新书** 按钮
2. 会出现一张书籍卡片，填写：
   - **书名**：输入书的名字（必填！不填无法输入图书价格哦）
   - **分类**：选择类型（漫画、诗歌、艺术等）
   - **店长**：默认是你的名字（可以改）
   - **货币**：选择原价的货币（日元、台币、澳币等）
   - **价格**：输入这本书的原价
   - **⭐推荐**：如果是店长推荐书，打开这个开关

3. 填完后，软件**自动计算**售价

### 情况2️⃣：卖 **二手书**

1. 点击 **♻️ 二手** 按钮
2. 和新书一样填写信息
3. 默认货币是澳元(AUD)
4. 输入这本二手书的售价（直接就是卖价5刀，不会再计算）

### 情况3️⃣：出租 **书阁的书**

1. 点击 **🏛️ 书阁** 按钮
2. 填写书名、分类、店长
3. 输入书阁主人自定价格（澳元）
4. 会有"🏛️ 书阁"标记


## ➕ 第四步：一次卖多本书？

**很简单！**

- 每卖一本，就点一次"新书"、"二手"或"书阁"按钮
- 软件会自动累加，下面会显示：
  - **会员价格**（左边大字）
  - **标准价格**（右边小字）

例如：
```
会员价格          标准价格
$35.50            $42.00
```

## 💾 第五步：保存记录

**重要！** 录完所有书后，一定要点底部的：

### ✓ 确认售出并保存

点完之后：
- ✅ 记录保存成功
- 📝 所有书籍清空
- 🎉 可以开始下一单了

### ⚠️ 注意事项

- **没保存就关软件** → 会弹出提醒，可以选择"保存并退出"
- 底部有红色提醒 **"📌 3 本书待保存"** → 记得点保存！
- 绿色按钮会**闪烁** → 提醒你还没保存

---

## 📊 第六步：查看销售记录（可选）

想看今天卖了多少？点右上角的 **📊** 按钮

### 你会看到：

**左边 - 销售明细**
- 每一单的详细信息
- 时间、书名、价格

**右边 - 数据统计**
- 总订单数
- 总书籍数
- 总收入
- 热门分类
- 店长业绩

### 🔍 搜索功能

- 输入书名或店长名字
- 快速找到某条记录

### 📥 导出Excel

1. 点击 **"📥 导出Excel"** 按钮
2. 选择导出范围：
   - 全部记录
   - 最近7天
   - 最近30天
   - 本月
3. 输入文件名
4. 点击"确认导出"
5. 选择保存位置

## 🔄 其他常用功能

### 切换店长

**场景：** 你值完班，下一个店长接班

1. 点击右上角的 **🔄** 按钮
2. 选择或输入新店长的名字
3. 点击"确认"

### 刷新汇率

**场景：** 汇率变了，需要更新

- 点击右上角的 **↻** 按钮
- 等几秒，软件会自动联网更新汇率
- 看到"汇率已更新"就好啦

### 删除某本书

**场景：** 录错了，想删掉

- 每张书籍卡片右边有个 **✕** 按钮
- 点一下就删除了

## ❓ 常见问题解答

### Q1: 为什么价格显示是 $0.00？

**A:** 可能是：
- 还没输入书名（书名必填）
- 还没输入价格
- 汇率还没更新（点一下刷新按钮 ↻）
### Q1: 为什么价格显示是 $3500000.00？

**A:** 可能是：
- 没联网哦
- 汇率还没更新（必须显示xx:xx分更新）

### Q2: 我不小心关了软件，数据会丢吗？

**A:** 
- 如果**已经保存** → 数据不会丢
- 如果**没保存** → 软件会提醒你，可以选择保存后再退出

### Q3: "推荐"开关是什么意思？

**A:** 
- **打开** → 这本书会员价更优惠（用推荐公式计算）
- **关闭** → 普通定价

### Q4: 二手书和新书有什么区别？

**A:**
- **新书** → 输入原价和货币，软件自动算售价
- **二手书** → 直接输入售价（澳元），不会再计算

### Q5: 我想改某本书的信息怎么办？

**A:** 
- 直接在卡片上修改
- 价格会自动重新计算
- 不用删掉重新添加
